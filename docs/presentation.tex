\section{Executive Summary}

\begin{infobox}
\textbf{Auto-Analyst} is an AI-powered demand planning dashboard for Careem Grocery that simulates a multi-agent executive War Room. Three AI personas --- \textcolor{CareemGreen}{\textbf{Careem}} (S\&OP Lead), \textcolor{RashidGold}{\textbf{Rashid}} (CFO), and \textcolor{NoorBlue}{\textbf{Noor}} (Ops Lead) --- analyze live supply chain data through their domain lens, each with an Emirati voice and region-inspired proverbs. The platform generates charts and interactive Dubai store maps on command via function-calling tools, detects anomalies, runs scenario simulations, and presents RAG scorecards across five dark stores. Agents autonomously consult each other through a \texttt{CONSULT} mechanism, creating a collaborative decision environment that mirrors how a real S\&OP team operates.
\end{infobox}

\subsection{How Agents Cooperate}
\begin{itemize}[leftmargin=*]
  \item \textbf{@mention override}: Type \texttt{@rashid} or \texttt{@noor} in chat to redirect your question to a specific agent
  \item \textbf{Autonomous consultation}: Careem ends his response with \texttt{[CONSULT: rashid]} --- the system automatically routes context to that agent, who responds in a separate colored chat bubble
  \item \textbf{Domain guardrails}: Each agent defers to the others when asked outside their expertise --- \textit{``That's Careem's turf, partner.''}
\end{itemize}

\end{tcolorbox}

\subsection{Data Flow}
\begin{enumerate}[leftmargin=*]
  \item \textbf{Data ingestion}: Synthetic 6-month grocery dataset (or uploaded CSV) loaded via Pandas
  \item \textbf{Analysis pipeline}: 8 analysis modules compute stats, forecasts, inventory health, risk flags, scorecards, anomalies
  \item \textbf{Context snapshot}: Aggregated analysis serialized into a compact text context for the LLM
  \item \textbf{Agent loop}: User message $\rightarrow$ selected agent's system prompt + context $\rightarrow$ LLM API call $\rightarrow$ response + optional tool calls (charts, maps) + optional \texttt{[CONSULT:]} triggers $\rightarrow$ consulting agent responds
  \item \textbf{Rendering}: Text, Altair charts, and Folium maps displayed inline in color-coded chat bubbles
\end{enumerate}
